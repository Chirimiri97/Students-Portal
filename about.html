Student Portal

A dashboard for students where they can organize their work at a single place. Calculator, ToDo List, Notes, Clock... Its not only this, there's more to come. (See here)

Functionalities -

  1. Calculator
  2. ToDo List
  3. Clock
  4. Notes

Technologies Used -

  • HTML
  • CSS
  • Javascript

Tools Used -

  1. Paper Css - https://www.getpapercss.com/
  2. Font Awesome Icons - https://fontawesome.com/
  3. Flaticon Image Icons - https://www.flaticon.com/authors/freepik

Whats more to come?

Right now it has a limited functionality. But in future I want add a databse to store the ToDo's, Notes (for now it is stored in Local Storage). I want to add certain API's such as Wikipedia, Dictionary, Language Translator etc. Will add some games like tic-tac-toe and Snake game.

The inspiration for this project came from here - https://github.com/FirePlank/Student-Portal
